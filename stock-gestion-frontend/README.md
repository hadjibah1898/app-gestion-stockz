# Stock Gestion Frontend

Application React pour la gestion de stock et de ventes.

## Fonctionnalités

- Authentification (Admin/Gérant)
- Dashboard avec statistiques
- Gestion des boutiques (Admin)
- Gestion des gérants (Admin)
- Gestion des articles
- Gestion des ventes (Gérant)
